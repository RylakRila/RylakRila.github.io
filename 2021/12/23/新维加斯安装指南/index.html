新维加斯安装指南

前言

该指南用于备份和提醒有关《辐射：新维加斯》安装的一些简单步骤和注意事项，同时存档一些有用的链接。

在2021年以及之后的时光里，想把新维加斯优化成能玩的游戏，肯定需要大量的步骤，其中也有大量的坑，本博客只能记录本人所遇到的情况以及本人的一些经验。

以后如果遇到新的坑也会在本博补充说明。

安装

以下记录较为详细的步骤

ENB的安装

别用ENB了，用ReShade吧。

前往Home (reshade.me)，点第一个绿色按钮下载就行。安装的时候记得选DirectX 9和新维加斯。具体可参考这个视频 How to install ReShade in Fallout New Vegas - YouTube

然后去Nexus搜索ReShade，找个你喜欢的预设（.ini）。进游戏按Home键进入面板，加载预设就行。

优化、前置的安装

https://www.bilibili.com/read/cv13735215 遵循该指南，安装必备的优化插件。

由于该教程内提供的文件已经过时，建议是根据里面提供的mod列表自行去N网上下载安装 (2022/3/30)。

推荐使用Mod organizer 2，请自行学习该工具的安装教程，教程很多这里不再赘述。

（举一反三能力强的，随便找个B社其他游戏的mo2教程都能学个七七八八）

注意事项:

  • 最好是能遵循该指南中“手动安装“的部分进行安装。
  • 如果要打汉化，请记得将4GB补丁在汉化后打上。
  • 这些插件（除了GECK）基本都是必要的，装上总没错。

这里列举一下需要放在游戏根目录而不是用MO管理器安装的插件：

  • xNVSE（就像是skse for skyrim, F4SE for fallout 4一样的东西，所以也要记得用这个启动游戏）
  • ogg Vorbis Libraries V6 （音频压缩优化插件）
  • 4gb补丁（记得在汉化后打上）
  • New Vegas Heap Replacer 3.2
  • GECK (如果你想当modder，可以玩玩这个，否则可以不管)

对于以上插件，如果必要，请做好备份工作

在以上的mod列表的基础上，此处再提供一些不少mod会用到的前置

  • Showoff NVSE https://www.nexusmods.com/newvegas/mods/72541

扩展了nvse的功能，不少mod也会用到这个插件。

  • MCM BugFix 2 https://www.nexusmods.com/newvegas/mods/42507?tab=files

修复了一些MCM菜单mod的bug，和mcm菜单在一个N网页面里。

汉化

该指南只赘述对于打MOD更有利的混合汉化类型

三种汉化的优劣（2021/12/23）：

  • 民间汉化：汉化完全，但是新地点和新任务乱码
  • 天邈汉化：基本解决乱码问题，但是汉化不完全（DLC没汉化）
  • 混合方案：民间汉化文本+天邈字库字体，有概率终端机乱码
  • 混合方案2：替换binkw32.dll后的方案，终端机乱码基本解决，但是新地点新任务不显示文字。

汉化下载：https://www.bilibili.com/read/cv9716268

以下介绍混合方案：

  1. 备份FalloutNV.exe和FalloutNVLauncher.exe文件

  2. 先运行安装天邈汉化，

    以下列举天邈在根目录产生的文件

    • CN_Uninstall文件夹用于卸载
    • 替换了游戏原来的binkw32.dll，并在根目录创建Backup文件夹放原版binkw32.dll
    • usub文件夹放着汉化组logo文件，主菜单logo汉化视频，以及字库（?)

    除此之外，天邈汉化还会在Data目录生成FalloutNVCN.esp、PreoderCN.esp、GunRunnersArsenalCN.esp这三个esp文件。

  3. 安装民间汉化

    民间汉化会替换data文件夹下所有的.esm文件、生成一个Texture和Meshes文件夹用于ui汉化。同时还会在根目录生成一个FNVTools.exe的文件，可以删除，前者的esm文件请做好英文原版备份以便恢复。

    在民间汉化的安装的最后一步，不要勾选下面两个插件，因为我们在优化前置那一步已经安装过了。

    该汉化替换了FalloutNV.exe和FalloutNVLauncher.exe

  4. 删除DATA文件夹下的FalloutNVCN.esp、PreoderCN.esp、GunRunnersArsenalCN.esp。或者不在MOD管理器里勾选。这三个是天邈的汉化esp。而混合方案应该暂时用民间汉化文本。

  5. 把第一步备份的exe文件复制回根目录覆盖。

  6. 解决终端机乱码及电台字幕不可用方法（如下图）：

    该方法似乎能一并解决左上角的汉化组logo

推荐MOD

Yukigachi unofficial patch

https://www.nexusmods.com/newvegas/mods/51664

非官方补丁（之一），修复了不少bug，且经过测试基本上没有和之前的优化插件冲突。

目前更新到12.3（2021/12/23），唯一的问题是目前没有特别完美的汉化，大家可以尝试利用ESP-ESM Translator打开该补丁里的esp和esm，然后用民间汉化的esm和N网的其他汉化补丁覆盖汉化（Translation - Load an already translated mod），接着自行把一些未覆盖到的字段看着已经汉化的本体esm对照汉化。

Diagonal Movement

https://www.nexusmods.com/newvegas/mods/64333

给第三人称邮差添加斜着走路的动画

ySI - Sorting Ycons

https://www.nexusmods.com/newvegas/mods/74358

在哔哔小子和其他UI界面里，按照物品类别给不同种类的物品前面放上图标并且按照图标排序，更便于管理自己的物品，捡垃圾必备。

需要前置yUI https://www.nexusmods.com/newvegas/mods/74357

Just Assorted Mods

https://www.nexusmods.com/newvegas/mods/66666

动态准心，命中提示，狙击开镜憋气，冲刺，武器轮盘，子弹时间等功能集为一体的mod。

汉化作者弃坑了，使用需要基础英语水平（

Casino Unbanner

https://www.nexusmods.com/newvegas/mods/35618

不让赌场永久禁止你赌博，可以用物品栏里的Casino Unbanner设置你超过赌场上限后，休息多久能重新再赌钱。

B42 Quickthrow

https://www.nexusmods.com/newvegas/mods/66686

拿着其他武器的同时扔手雷

NMC's Textures NV MEDIUM Pack

https://www.nexusmods.com/newvegas/mods/43135

高清化补丁1

有两个文件

Ojo Bueno Texture Pack

https://www.nexusmods.com/newvegas/mods/39755

高清化补丁2

Simplified WRP

一个WRP（https://www.nexusmods.com/newvegas/mods/38285）的精简版本，不需要担心汉化问题就能有WRP里的高清武器材质。
